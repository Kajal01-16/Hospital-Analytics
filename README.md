Smart Hospital Emergency Room Analytics System
Overview
This project is a Streamlit-based Hospital Emergency Room Analytics System developed for AI, ML, Data Science, and Data Analytics. It helps hospitals manage patient data, prioritize treatment, analyze workloads, forecast resources, and generate reports.
Objectives
•	Manage patient records
•	Validate healthcare data
•	Calculate patient priority scores
•	Classify patients by urgency
•	Analyze queue, doctors, beds, and peak hours
•	Generate visualizations and Excel reports
•	Provide AI-based recommendations
Features
1.	Dataset generation (1000+ records)
2.	Data validation
3.	Priority algorithm
4.	Patient classification
5.	Queue management
6.	Doctor requirement analysis
7.	Bed occupancy prediction
8.	Peak hour detection
9.	Statistical analysis
10.	Emergency alert system
11.	Charts and visualizations
12.	Resource forecasting
13.	KPI dashboard
14.	Excel report generation
15.	AI recommendation system
16.	Streamlit web dashboard

Installation
1.	Install Python 3.10+.
2.	Install dependencies: pip install -r requirements.txt
3.	Generate the dataset: python dataset_generator.py
4.	Run the application: streamlit run app.py
